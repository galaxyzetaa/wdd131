Goob Club Site Plan

Site Name: Goob Club

Reason

This is acting as the personal website for Goob from Meet the Robinson's. Since he is young and showcasing his personality, I wanted a site name that was short and catchy.

Purpose

This is Goob's personal portfolio as he seeks friends and adoptive parents. Viewers will look at Goob's portfolio and fill out a contact form if they are interested in either position.

Scenarios

-Is Goob the type of kid I would want to be friends with?

-Is Goob the type of kid I would want to adopt?

Color Schema

I based the color scheme off Goob's baseball team.

Primary Color: #d6d8d6;

This color will be used for the main background, as well as the font color for the header area.

Secondary Color: #306939;

This color will be used for the header and accents.

Accent Color: black

This color will be used as the main font color in the main section, as well as accents.

Typography

Heading Font: Seymour One

Paragraph Font: Merriweather
